### コンパイル

コンパイルにはLuaLaTeXが必要です。[TeXLive](https://www.tug.org/texlive/)（または[MacTeX](https://tug.org/mactex/)）をインストールしてください。
OSXでは次のコマンドでアニメーションあり・なし両方のスライドを生成することができます。

```
$ make all
```
